# 🛒 Lista de la Compra PWA

**Lista de la Compra** es una aplicación web moderna, rápida y sencilla diseñada para ayudarte a gestionar tus compras diarias de manera eficiente. Gracias a su tecnología como Progressive Web App (PWA), puedes usarla tanto en tu navegador como instalarla en tu dispositivo móvil para usarla sin conexión.

## ✨ Características principales

- 📝 **Gestión de productos**: Añade productos a tu lista de forma rápida.
- 🌓 **Modo Oscuro/Claro**: Adaptado para una visualización cómoda en cualquier momento del día.
- 🔢 **Contador en tiempo real**: Visualiza cuántos productos tienes en tu lista actualmente.
- 📱 **PWA (Progressive Web App)**: Instálala en tu pantalla de inicio como una aplicación nativa.
- 🔌 **Funcionamiento Offline**: Accede a tu lista incluso cuando no tengas conexión a Internet.
- 🔔 **Notificaciones (Toasts)**: Avisos visuales al realizar acciones.

## 🚀 Tecnologías utilizadas

- **HTML5**: Estructura semántica.
- **CSS3**: Diseño moderno y responsive con variables CSS para el cambio de temas.
- **JavaScript (Vanilla)**: Lógica de la aplicación sin dependencias externas.
- **Service Workers**: Para el soporte offline y almacenamiento en caché.
- **Manifest JSON**: Configura la apariencia de la PWA.

## 📂 Estructura del Proyecto

```text
├── index.html        # Página principal
├── css/
│   └── main.css      # Estilos y temas (claro/oscuro)
├── js/
│   └── app.js        # Lógica de la aplicación
├── img/              # Iconos e imágenes
├── manifest.json     # Configuración PWA
└── sw-listacompra.js # Service Worker para modo offline
```

## 📲 Instalación

1. Accede a la URL de la aplicación desde tu navegador móvil (Chrome o Safari).
2. Selecciona la opción **"Añadir a la pantalla de inicio"** o el icono de instalación en la barra de direcciones.
3. ¡Listo! Ya tienes tu lista de la compra siempre a mano.
